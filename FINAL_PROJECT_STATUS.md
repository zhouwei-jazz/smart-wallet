## 🔧 技术架构

### 前端技术栈
- **Web**: Next.js 16.1 + React 19.2 + TypeScript 5.9
- **Mobile**: Expo 54 + React Native + TypeScript 5.9
- **UI**: Tailwind CSS + 现代玻璃态设计
- **状态管理**: TanStack Query v5 + React Context

### 后端技术栈
- **数据库**: Supabase PostgreSQL
- **认证**: Supabase Auth + OAuth (GitHub/Google)
- **实时**: Supabase Realtime
- **存储**: Supabase Storage (准备就绪)
- **API**: PostgREST 自动生成

### 架构特点
- **Monorepo**: pnpm workspaces
- **共享代码**: packages/core (类型、API、Hooks)
- **类型安全**: 100% TypeScript 覆盖
- **实时同步**: WebSocket 多端同步
